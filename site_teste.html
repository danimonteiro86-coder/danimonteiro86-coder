Bem-vindo ao TesteFácil!

Esta plataforma foi desenvolvida para ajudar estudantes de teste de software a praticarem suas habilidades em um ambiente controlado.

Como usar:

  1. Acesse a aba "Sistema para Testar" para interagir com um sistema fictício que contém defeitos intencionais.
  2. Na aba "Casos de Teste", documente os casos de teste que você planeja executar.
  3. Execute os testes e quando encontrar um defeito, registre-o na aba "Registrar Defeito".
  4. Acompanhe todos os defeitos encontrados na aba "Lista de Defeitos".

Sistema de Cadastro de Usuários - Requisitos

O sistema permite cadastrar usuários com as seguintes regras:

  • Nome deve ter pelo menos 2 caracteres
  • Email deve ter formato válido
  • Senha deve ter pelo menos 6 caracteres
  • Idade deve ser entre 18 e 120 anos
  • Todos os campos são obrigatórios

Desafio: Encontre pelo menos 5 defeitos neste sistema!

Dica: Os dados são salvos apenas na memória do navegador e serão perdidos ao recarregar a página.
